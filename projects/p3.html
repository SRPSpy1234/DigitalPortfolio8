Project Description

For the code generation portion of this project, I coded a program that displays 7 random advertisements on a 16x2 LCD screen when a button is pressed. I made my advertisments use the brand name and the slogan of that brand. I used the LCD library to make screen function correctly. I had only one constant variable, which was the button pin. I had two variable to store the button state and the prev button state. I also created a variable named 'reply' that didn't have a value until the random function was used to give it a value 1-7. I used one if statement to check if the button state was not equal to the previous button state and when that was true, another if statement checked if the button state was low.

For the build portion of this project, I used a 16x2 LCD screen, a push button, and an R4 Arduino board. I connected the LCD to the breadboard with some wires and then jumped those wires to the arduino pins. We mostly used digital pins for this project, as the only analog value we had was the potentiometer, which connected directly to the LCD screen at pin V0. The button was connected to the breadboard and then jumped to pin 12 on the arduino. We used both a 10k ohm resistor and a 220 ohm resistor for this project. We used them to create a pull down resistor for the button.

During this project, we encountered a few minor issues. One of them was that after being built, the LCD displayed random characters instead of the advertisements. It looked to be some sort of ancient script and it was unreadable.
