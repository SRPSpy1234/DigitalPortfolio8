Project1_1: Blink

12/4/25

#include "Arduino_LED_Matrix.h"

ArduinoLEDMatrix matrix;

// Frame with all LEDs on (12x8 matrix)
const uint32_t allOn[] = {
  0xffffffff,
  0xffffffff,
  0xffffffff
};

// the setup function runs once when you power the board
void setup() {
  // starts the LED matrix
  matrix.begin();
}

void loop() {
  // turn on all LEDs in the matrix (HIGH)
  matrix.loadFrame(allOn);
  
  // wait for 500ms
  delay(500);
  
  // turn the LEDs off by clearing the matrix (LOW)
  matrix.clear();
  
  // wait for 500ms
  delay(500);
}

Project Description

For the first project with Arduino, we created a simple block of code that would make the pin 13 LED blink on and off at half second intervals. This project helped us get used to the Arduino IDE and basic programming concepts such as setting pin modes and using delays. We also learned about the structure of an Arduino sketch, including the void setup() and void loop() functions. I also experimented with changing the blink rate and used my Arduino R4 Wifi to test the 12x8 LED matrix display.

Project Description

For the second part of Project 1, I experimented with the Arduino R4 Wifi and added a simple LED circuit. This project helped me experiment with resistors in circuits to limit current flow and protect components.

Project1_3: Button

Project Description

For the third part of Project 1, I used a button to control an LED. The button was connected to the positive rail with a jumper wire, and the other side was connected to the positive leg of the LED. I had a 220-ohm resistor connected to the positive rail between the jumper cable and the button. The negative leg of the LED was connected to the ground rail with another jumper wire. When the button gets pressed, the circuit closes and the LED lights up. After, I also added a blink feature to the LED.

// Define LED pins
const int RED_LED = 13;
const int YELLOW_LED = 12;
const int GREEN_LED = 11;

void setup() {
  pinMode(RED_LED, OUTPUT);
  pinMode(YELLOW_LED, OUTPUT);
  pinMode(GREEN_LED, OUTPUT);

  // Start with all LEDs BUT GREEN ON
  digitalWrite(RED_LED, LOW);
  digitalWrite(YELLOW_LED, LOW);
  digitalWrite(GREEN_LED, HIGH);
}

void loop() {
  // GREEN ON (others OFF)
  digitalWrite(GREEN_LED, HIGH);
  digitalWrite(YELLOW_LED, LOW);
  digitalWrite(RED_LED, LOW);
  delay(9000);

  // YELLOW ON (others OFF)
  digitalWrite(GREEN_LED, LOW);
  digitalWrite(YELLOW_LED, HIGH);
  digitalWrite(RED_LED, LOW);
  delay(9000);

  // RED ON (others OFF)
  digitalWrite(GREEN_LED, LOW);
  digitalWrite(YELLOW_LED, LOW);
  digitalWrite(RED_LED, HIGH);
  delay(9000);
}

Project Description

For the final part of Project 1, we took all of our previous knowledge of arduinos, soldering and power tool and combined them to create a final project that encompassed everything we learned. The goal of the project was to create a Trafic Light. The traffic light would have red, yellow and green LEDs that would alternate every 9 seconds. The box to hold the components was built using 5.5" x 1" pieces of wood that were cut then screwed together. The front panel of the box was attached using velcro so it could be easily removed to access the circuit inside. We faced a few chanllenges during the project, such as accidently making our housing to deep. This entire project helped us complete our first full Arduino circuit, code and housing.
